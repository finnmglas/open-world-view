# open-world-view

open-world-view is a nextjs frontend and python backend application, presenting real-time geo datasets in an unprecedented way.

open-world-view is inspired by the recent work of Ex-Google PM.

## Installation

Clone this repository:

```sh
git clone https://github.com/finnmglas/open-world-view
```

Use docker to securely run the world view.

```sh
docker compose up -d
```

Then the app is available at:

[http://localhost:3000/](http://localhost:3000/)

## Usage

Tap around, scroll around, have fun!
